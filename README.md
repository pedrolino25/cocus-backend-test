#  COCUS Backend Test

API to list all user github repositories with the option to include or exclude forked repositories

---

## Tech Stack
- Node.js
- TypeScript
- Express
- Swagger
- Jest

## Installation
npm install

## Environment Variables
Create a .env file:
PORT=3000

## Run the App
Development:
npm run dev

Production:
npm run build
npm start

## API Documentation
Swagger UI:
http://localhost:3000/api-documentation

## Testing
npm test
